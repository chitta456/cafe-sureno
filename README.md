# ☕ Café Sureño

Sitio web para una cafetería de especialidad patagónica. Proyecto desarrollado como práctica de HTML y CSS puro, sin frameworks ni librerías.

🔗 **[Ver demo en vivo](https://chitta456.github.io/cafe-sureno)**

---

## 📸 Preview

![Preview de Café Sureño](screenshot.png)

---

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 (Flexbox, Grid, Variables CSS, Media Queries)
- Google Fonts (Playfair Display + Lato)

---

## ✨ Funcionalidades

- Header sticky con navegación por anclas
- Hero con imagen de fondo y gradiente
- Grilla de productos con cards (nombre y precio)
- Sección "Sobre nosotros"
- Formulario de contacto
- Diseño responsive (mobile y desktop)
